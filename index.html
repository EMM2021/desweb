Introducción al Desarrollo Web

Comprende los fundamentos y la arquitectura de las aplicaciones web modernas

¿Qué son las Aplicaciones Web?

Las aplicaciones web son programas que se ejecutan en navegadores web y pueden ser accedidas desde cualquier dispositivo conectado a internet. No requieren instalación y se actualizan automáticamente.

  • Accesibles desde cualquier dispositivo
  • Actualizaciones automáticas
  • Multiplataforma
  • Fácil mantenimiento

Frontend vs Backend

Frontend (Cliente)

La parte visible de la aplicación que interactúa directamente con el usuario. Incluye la interfaz, diseño, navegación y experiencia de usuario.

Backend (Servidor)

La lógica del servidor, bases de datos, autenticación, APIs y toda la funcionalidad que procesa datos y gestiona la aplicación desde el servidor.

Fundamentos Esenciales

HTML5

Lenguaje de marcado que estructura el contenido web. Define elementos, semántica y accesibilidad.

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi Web</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
  </body>
</html>

CSS3

Controla el diseño, colores, tipografía y animaciones. Incluye Flexbox, Grid y responsive design.

.container {
  display: flex;
  justify-content: center;
  align-items: center;
  background: linear-gradient(
    45deg, #3b82f6, #8b5cf6
  );
  border-radius: 10px;
}

JavaScript

Lenguaje de programación que añade interactividad, maneja eventos y comunica con APIs.

// Función interactiva
function saludar(nombre) {
  const mensaje = `¡Hola, ${nombre}!`;
  document.getElementById('saludo')
    .textContent = mensaje;
}

// Event listener
button.addEventListener('click', () => {
  saludar('Desarrollador');
});

Frameworks y Librerías Modernas

React

Librería de Facebook para crear interfaces de usuario con componentes reutilizables y estado virtual DOM.

  • Componentes reutilizables
  • Virtual DOM rápido
  • Ecosistema extenso
  • Gran comunidad

Vue.js

Framework progresivo con curva de aprendizaje suave, perfecto para proyectos de cualquier tamaño.

  • Fácil de aprender
  • Progresivo
  • Documentación excelente
  • Flexible

Angular

Framework completo de Google con TypeScript, perfecto para aplicaciones enterprise complejas.

  • Framework completo
  • TypeScript nativo
  • Escalable
  • Herramientas integradas

TypeScript

Superset de JavaScript que añade tipado estático, mejorando la detección de errores y la experiencia de desarrollo.

interface Usuario {
  id: number;
  nombre: string;
  email: string;
}

function crearUsuario(datos: Usuario): Usuario {
  return { ...datos, activo: true };
}
  • Detección temprana de errores
  • Mejor IDE support
  • Refactoring seguro

Tecnologías Backend

Explora los lenguajes y frameworks que potencian el servidor

JavaScript/Node.js

Permite usar JavaScript en el servidor. Event-driven, no bloqueante, perfecto para aplicaciones en tiempo real.

const express = require('express');
const app = express();

app.get('/api/usuarios', (req, res) => {
  res.json({ usuarios: [] });
});

app.listen(3000);
  • Mismo lenguaje frontend/backend
  • NPM ecosystem
  • Alta performance I/O

Python

Lenguaje versátil con sintaxis clara. Django, Flask y FastAPI ofrecen diferentes enfoques de desarrollo.

# FastAPI ejemplo
from fastapi import FastAPI

app = FastAPI()

@app.get("/usuarios")
def obtener_usuarios():
    return {"usuarios": []}
Django Flask FastAPI

Java

Lenguaje robusto y escalable. Spring Boot facilita la creación de aplicaciones enterprise.

@RestController
public class UsuarioController {

  @GetMapping("/usuarios")
  public List<Usuario> obtenerUsuarios() {
    return usuarioService.findAll();
  }
}
  • Fuertemente tipado
  • Ecosistema maduro
  • Spring Boot

PHP

Lenguaje web tradicional, fácil de aprender. Laravel y Symfony son frameworks modernos y potentes.

// Laravel Route
Route::get('/usuarios', function () {
    return Usuario::all();
});

// Controller
class UsuarioController {
    public function index() {
        return view('usuarios');
    }
}
Laravel Symfony

C# (.NET)

Lenguaje de Microsoft, multiplataforma con .NET Core. Excelente para aplicaciones empresariales.

[ApiController]
[Route("api/[controller]")]
public class UsuariosController : ControllerBase
{
    [HttpGet]
    public ActionResult<IEnumerable<Usuario>> Get()
    {
        return Ok(usuarios);
    }
}
  • Fuertemente tipado
  • Cross-platform
  • Alto rendimiento

Ruby

Lenguaje elegante con Ruby on Rails, framework que prioriza la convención sobre configuración.

# Rails Controller
class UsuariosController < ApplicationController
  def index
    @usuarios = Usuario.all
    render json: @usuarios
  end
end

# Route
resources :usuarios
  • Sintaxis elegante
  • Rails framework
  • Desarrollo rápido

Go

Lenguaje de Google, compilado, concurrente y eficiente. Ideal para microservicios y sistemas distribuidos.

func main() {
  http.HandleFunc("/usuarios", usuariosHandler)
  log.Fatal(http.ListenAndServe(":8080", nil))
}

func usuariosHandler(w http.ResponseWriter, r *http.Request) {
  json.NewEncoder(w).Encode(usuarios)
}
  • Alta performance
  • Concurrencia nativa
  • Compilación rápida

Rust

Lenguaje de sistemas memory-safe, extremadamente rápido y seguro. Ideal para servicios de alto rendimiento.

use actix_web::{web, App, HttpResponse, HttpServer};

async fn usuarios() -> HttpResponse {
    HttpResponse::Ok().json(vec!["usuario1", "usuario2"])
}

#[actix_web::main]
async fn main() -> std::io::Result<()> {
    HttpServer::new(|| App::new().route("/usuarios", web::get().to(usuarios)))
        .bind("127.0.0.1:8080")?.run().await
}
  • Memory safety
  • Zero-cost abstractions
  • Ultra performance

Bases de Datos

Comprende las diferencias entre SQL y NoSQL, y elige la mejor opción para tu proyecto

Bases de Datos SQL

Bases de datos relacionales con estructura de tablas, esquemas fijos y lenguaje SQL estándar.

Características

  • • Estructura rígida (esquemas)
  • • Relaciones entre tablas
  • • ACID compliance
  • • Lenguaje SQL estándar

Ventajas

  • • Consistencia de datos
  • • Transacciones robustas
  • • Consultas complejas
  • • Madurez y estabilidad

Bases de Datos NoSQL

Bases de datos no relacionales, flexibles, escalables horizontalmente y sin esquemas fijos.

Características

  • • Esquemas flexibles
  • • Escalabilidad horizontal
  • • Diversos modelos de datos
  • • Alto rendimiento

Ventajas

  • • Flexibilidad de esquemas
  • • Escalabilidad masiva
  • • Desarrollo ágil
  • • Manejo de big data

MySQL

SQL

Base de datos relacional más popular del mundo. Rápida, confiable y ampliamente soportada.

CREATE TABLE usuarios (
  id INT AUTO_INCREMENT PRIMARY KEY,
  nombre VARCHAR(100) NOT NULL,
  email VARCHAR(100) UNIQUE,
  fecha_registro TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

SELECT * FROM usuarios WHERE activo = 1;
  • Gratuita y open source
  • Rendimiento excelente
  • Amplio soporte

PostgreSQL

SQL

Base de datos objeto-relacional avanzada con características enterprise y extensibilidad.

-- Tipos de datos avanzados
CREATE TABLE productos (
  id SERIAL PRIMARY KEY,
  datos JSONB,
  tags TEXT[],
  ubicacion POINT
);

-- Consulta JSON
SELECT * FROM productos
WHERE datos->>'categoria' = 'electronica';
  • Muy extensible
  • Tipos de datos avanzados
  • ACID compliant

MongoDB

NoSQL

Base de datos de documentos JSON, flexible y escalable. Perfecta para aplicaciones modernas.

// Insertar documento
db.usuarios.insertOne({
  nombre: "Juan Pérez",
  email: "juan@email.com",
  perfil: {
    edad: 30,
    intereses: ["programación", "música"]
  }
});

// Consultar
db.usuarios.find({"perfil.edad": {$gte: 25}});
  • Esquemas flexibles
  • Escalabilidad horizontal
  • JSON nativo

ORMs Populares (Object-Relational Mapping)

SQLAlchemy

Hibernate

ORM estándar para aplicaciones Java

Control de Versiones - Git

Sistema de control de versiones distribuido, esencial para colaboración y gestión de código.

# Comandos básicos de Git
git init # Inicializar repositorio
git add . # Agregar archivos al staging
git commit -m "mensaje" # Confirmar cambios
git push origin main # Subir cambios al repositorio remoto
git pull origin main # Descargar últimos cambios
git branch feature/nueva # Crear nueva rama
git checkout feature/nueva # Cambiar a rama
git merge feature/nueva # Fusionar rama

GitHub

Plataforma de hosting más popular

GitLab

CI/CD integrado
